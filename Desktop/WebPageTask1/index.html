HTML – Structure of Webpage

HTML stands for HyperText Markup Language. It is used to create the structure of webpages using headings, paragraphs, images and links.

Heading Tags

This is representation of Heading tags

the tag used for heading is <h1> to <h6>

Heading tags are used to define titles and subtitles on a webpage.
HTML provides six heading tags from <h1> to <h6>.
<h1> is the largest heading and <h6> is the smallest heading.

Paragraph Tag

This is representation of paragraph tag

The paragraph tag is used to write text content on a webpage.
It is written using the <p> tag.
Paragraph tags help organize text into readable sections.

Image tag

This is representation of image tag

The image tag is used to display images on a webpage.
It is written using the <img> tag.
The <img> tag requires the 'src' attribute to specify the image source and the 'alt' attribute for alternative text.

Anchor Tag

This is representation of anchor tag

The anchor tag is used to create hyperlinks on a webpage.
It is written using the <a> tag.
The <a> tag requires the 'href' attribute to specify the URL of the link.

CSS – Styling the Webpage

CSS stands for Cascading Style Sheets. It is used to style and design webpages. CSS helps change colors, fonts, spacing, alignment and layout to make webpages more attractive and user-friendly.

In this webpage, CSS is applied using an external stylesheet called style.css to improve the appearance of headings, paragraphs, images, buttons and overall layout.

Selector :
tells CSS which HTML element you want to style (e.g., p, h1, .class, #id).
Property :
defines the aspect of the element you want to change (e.g., color, font-size, margin).
Value :
specifies the setting for the property (e.g., red, 16px, 10px).
closing brace :
represents the end of a CSS rule.

JavaScript – Adding Interactivity

document.getElementById("elementID")DOM Access :
This line tells JavaScript to find an element in the HTML page by its unique id.
Example: If you have <button id="messageBtn">Click Me</button>, this code grabs that button.
.onclick = function()Event Handler :
This attaches an event listener to the element.
It means: “When the user clicks this element, run the following function.”
The function() { ... } block defines what happens when the click occurs.
alert("Message here");Alert Function :
Inside the function, this command shows a popup alert box with the text you specify.
Example: alert("Welcome!") will display a small popup saying “Welcome!” when the button is clicked.
closing brace :
This marks the end of the function block.
Everything between { ... } is the code that runs when the event happens.
